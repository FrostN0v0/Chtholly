## 🎉 使用

_✨ ChatGPT对话 ✨_

|    命令    |                         说明                         |        示例        |
|:--------:|:--------------------------------------------------:|:----------------:|
|  /talk   |                  开始对话，默认群里@机器人也可以                  | /talk 先有鸡还是先有蛋？  |
|   /hi    |                    沉浸式对话（仅限私聊）                     |                  |
|  /speak  |                      gpt转语音对话                      | /speak 先有鸡还是先有蛋？ |
|  /reset  |                    重置对话（不会重置预设）                    |                  |
| /prompt  |                 设置预设（人格），设置后会重置对话                  |                  |
| /chatgpt | 如果talk_with_chatgpt_all_group_enable为false，则用该命令启用 |                  |
|   修改角色   |                   设置预设vits转语音角色                    |    修改角色 <派蒙>     |
